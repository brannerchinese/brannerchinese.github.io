Data within literal curly brackets using Python format()

Permalink: 2013-10-20 20:43:00 by David Prager Branner in

In current Python syntax, the format() command for strings inserts data inside of curly brackets in a string:

>>> 'Insert five here: {}.'.format(5)
'Insert five here: 5.'

If you want literal brackets in a string modified by format(), you have to escape the brackets by doubling them:

{.python}

'Here is what brackets look like: {{}}.'.format() 'Here is what brackets look like: {}.'

double-escaped brackets for the literal brackets and then another pair
of brackets as the socket for the data:

{.python}
>>> 'Now put five inside of brackets: {{{}}}.'.format(5)
'Now put five inside of brackets: {5}.'
